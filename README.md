# 🏌️‍♂️ BirdieFund Elite — 4K Premium Golf & Impact Platform

Welcome to **BirdieFund Elite**, the high-fidelity golfing platform where every swing creates a legacy of impact. Designed for the discerning golfer, our platform combines 4K UHD visual excellence with a rigorous, PRD-compliant draw engine.

## 🌟 Elite Features

### 💎 4K Visual Mastery
*   **Ultra-HD Imagery**: High-fidelity 4K assets integrated across the landing page and member dashboards.
*   **Branded Experience**: Featuring custom-macro golf ball assets and sunrise coastal landscapes for a luxury-grade aesthetic.
*   **Dynamic UI**: Powered by **Framer Motion** for silky-smooth, premium transitions and interactions.

### ⚙️ The Elite Engine (PRD-Logic)
*   **The Rolling 5**: Only your latest 5 Stableford scores matter. A new submission automatically cycles out the oldest, keeping the draw entry pool dynamic.
*   **The Golden Split**: Every entry fee is transparently divided:
    *   **40%**: Direct Charity Funding.
    *   **35%**: Monthly Jackpot Accumulation.
    *   **25%**: Guaranteed Monthly Draw Winner.
*   **Verified Draws**: Integrated PostgreSQL RPCs for deterministic, fair prize distribution.

### 🕊️ Mission-Driven Impact
*   **Charity Spotlight**: Front-and-center mission updates showcasing where your contributions go.
*   **Member Mission Showcase**: An emotional dashboard experiences linking your performance to real-world coaching and facilities development.

## 🚀 Tech Stack

- **Core**: Next.js 16 (App Router)
- **Database/Auth**: Supabase (PostgreSQL)
- **Logic**: Custom SQL RPCs for Draw Generation
- **Styling**: Tailwind CSS 4 + Lucide Icons
- **Motion**: Framer Motion for premium kinetic UI
- **Deployment**: Vercel (Production Ready)

## 🛠️ Getting Started

1.  **Clone the Repo**
2.  **Install Dependencies**: `npm install`
3.  **Local Dev**: `npm run dev`
4.  **Production Build**: `npm run build`

## 📡 Live Environment
The platform is linked to Vercel (Project: `internship`).
To deploy the final Elite build, use:
```bash
vercel --prod
```
